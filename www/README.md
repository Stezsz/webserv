# webserv - Test Pages

Simple test pages for the webserv HTTP/1.1 server implementation.

## Structure

```
www/
├── index.html          # Main page with links to all tests
├── upload.html         # File upload test (POST)
├── delete.html         # File deletion test (DELETE)
├── cgi.html           # CGI execution test
├── cgi-bin/
│   └── test.py        # Python CGI script
├── errors/
│   ├── 403.html       # Forbidden error page
│   ├── 404.html       # Not found error page
│   ├── 405.html       # Method not allowed error page
│   └── 500.html       # Internal server error page
└── uploads/           # Directory for uploaded files
```

## Tests

### 1. File Upload (POST)
- **Page:** `/upload.html`
- **Tests:** POST with multipart/form-data
- **Expected:** File saved to `uploads/` directory

### 2. File Delete (DELETE)
- **Page:** `/delete.html`
- **Tests:** DELETE method
- **Expected:** File removed, 204 No Content status

### 3. CGI Execution
- **Page:** `/cgi.html`
- **Script:** `/cgi-bin/test.py`
- **Tests:** Python CGI script execution
- **Expected:** Dynamic HTML generated by Python

### 4. Directory Listing
- **URL:** `/uploads/`
- **Tests:** Autoindex on
- **Expected:** HTML listing of uploaded files

### 5. Static Files
- **Tests:** GET method for HTML, CSS, images
- **Expected:** Correct Content-Type headers

### 6. Error Pages
- **404:** Access non-existent file
- **403:** Access forbidden resource
- **405:** Use unsupported HTTP method
- **500:** Trigger server error

## Command Line Tests

```bash
# Upload file
curl -X POST -F "file=@test.txt" http://localhost:8080/upload

# Delete file
curl -X DELETE http://localhost:8080/uploads/test.txt

# CGI test
curl http://localhost:8080/cgi-bin/test.py

# Directory listing
curl http://localhost:8080/uploads/

# GET static file
curl http://localhost:8080/index.html
```

## Notes

- All pages use simple, clean HTML/CSS
- No external dependencies
- Focus on testing HTTP/1.1 features required by the subject
